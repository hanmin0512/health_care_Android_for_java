Health Care Application

## 사용한 라이브러리
-  Push Alarm
  > AlarmManager, NotificationManager, SharedPreferences
-  DataBase
  > SQLite
-  Calendar
  > RecylcerView로 calendar 구현
-  그래프
  > MPAndroidChart
-  GIF
  > Glide로 삽입
